# VA-Usage Code Release

This directory is the released code package linked from the paper page.

It contains two main parts:

- `agentic-extraction/`
- `next-analytic-move-benchmark/`

## `agentic-extraction/`

This directory contains the released materials for the agentic extraction method, including preprocessing scripts, released artifacts, extraction skills, and schemas.

Start here:

- `agentic-extraction/README.md`

## `next-analytic-move-benchmark/`

This directory contains the released `Next Analytic Move` benchmark package, including benchmark inputs, gold labels, runtime code, and evaluation code.

Start here:

- `next-analytic-move-benchmark/README.md`

## Notes

- `agentic-extraction/` focuses on how paper evidence is preprocessed and then extracted into structured specifications.
- `next-analytic-move-benchmark/` focuses on running and evaluating the released benchmark package.
- This release does not include benchmark construction scripts.
