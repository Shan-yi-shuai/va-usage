# Knowledge Base Subset

This directory contains the released structured outputs for the four showcase papers:

- `BDIViz`
- `ConceptViz`
- `DMap`
- `FluxFlow`

Each paper directory includes:

- `paper-usage-spec.json`
- `paper-workflow-spec.json`
- `system-spec.json`

These files are included so the released `agentic-extraction/` package remains self-contained for the four-paper showcase.
